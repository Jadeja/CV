Website accessibility

Access keys are keyboard shortcuts that provide links to important areas of the site. Windows users can press ALT + an access key for Google chrome browser. Apple Macintosh users can press CONTROL + an access key. This website uses the following standard access keys:

  • Access key C - Analysis Page
  • Access key I - Personal Information Page
  • Access key S - Personal Statement
  • Access key E - Education Section
  • Access key W - Work Experience Section
  • Access key B - Back Button
  • Access key L - Logout
